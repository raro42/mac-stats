# Draft outreach post (Mac / Apple Silicon communities)

**Title:** Lightweight menu-bar monitor for Apple Silicon (optional local LLM)

**Body (short):**

mac-stats is a free MIT menu-bar system monitor for Apple Silicon — think Stats/iStat-style CPU in the bar, glass window, themes, process list.

AI is **opt-in**: leave it off for a zero-config monitor. Turn on local Ollama (and optionally Discord) when you want an on-device agent.

```bash
brew tap raro42/mac-stats https://github.com/raro42/mac-stats
brew install --cask mac-stats
```

Apple Silicon only. No cloud telemetry — data stays in `~/.mac-stats/`.

Repo: https://github.com/raro42/mac-stats
