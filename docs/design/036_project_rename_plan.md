# Project Rename Plan

## Objective
The application is evolving from a clone of "mac-stats" to a dedicated local AI application. To reflect this shift, we need a new project and GitHub repository name. The new name should utilize a German noun (Substantiv) that reflects intelligence, assistance, and local context.

## Candiate Names
We have compiled a shortlist of 6 potential names for the new GitHub repository and application name. 

1. **`lokaler-verstand`** (Local Mind / Reason)
   - **Why:** Perfectly captures the concept of intelligence ("Verstand") running entirely locally. It is easily readable for non-German speakers ("lokal" ≈ "local") and perfectly suits standard `kebab-case` GitHub repo names.
   - **App Name:** Lokaler Verstand / LokalerVerstand

2. **`lokaler-lotse`** (Local Guide / Navigator)
   - **Why:** Focuses more on the "assistant" aspect. A "Lotse" navigates complex waters, similar to how an AI assistant helps navigate tasks.
   - **App Name:** Lokaler Lotse

3. **`lokaler-denker`** (Local Thinker)
   - **Why:** Very direct and active. Clearly states that this tool processes and thinks alongside the user.
   - **App Name:** Lokaler Denker

4. **`eigen-verstand`** (Own Mind)
   - **Why:** "Eigen" means "own", emphasizing that the user retains ownership and privacy of the AI's mind.
   - **App Name:** Eigen Verstand / EigenVerstand
   
5. **`mac-verstand`** (Mac Mind)
   - **Why:** Keeps a connection to its macOS roots ("mac-stats") while embracing the new AI-focused paradigm.
   - **App Name:** Mac Verstand

6. **`mein-hirn`** (My Brain)
   - **Why:** Plays into the "Second Brain" concept. The use of "mein" emphasizes user ownership, and "hirn" represents local processing power and memory. Very personal and direct.
   - **App Name:** Mein Hirn

## Next Steps
- Finalize the choice from the 6 candidates.
- Update repository configurations, `Cargo.toml`, `tauri.conf.json`, and `package.json` to reflect the chosen name.
- Update all relevant documentation and references pointing to the old "mac-stats" name.
